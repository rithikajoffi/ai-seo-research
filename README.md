# AI-Powered SEO Content Production — Research Repository

## Purpose
This repo collects primary source material from 10 practitioners actively using AI in SEO content workflows. The goal is to extract a repeatable playbook grounded in real tactics, not theory.

## Expert Selection Logic
I prioritized people who:
- Run agencies or SaaS products dependent on SEO
- Publish data from real campaigns
- Are actively experimenting with AI tools (not just writing about them)

## Repository Structure
- `/research/sources.md` — Annotated expert list
- `/research/linkedin-posts/` — Recent posts by author
- `/research/youtube-transcripts/` — Transcripts by video
- `/research/other/` — Podcast summaries, newsletters, etc.
